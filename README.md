# Daseeki Bags

A standalone bag interface for WoW Classic Era — combines all your bags into a single sortable, searchable view, with currency tracking and account-wide saved sets.

## Features

- Single combined bag window with sorting and search.
- Account-wide item sets and rules (`DaseekiBagsSets`).
- Currency tracker integrated into the bag frame.
- Pluggable skinning so it matches the rest of the Daseeki suite.
- Localized for multiple languages (see `core/localization`).

## Requires

- Optional: [Daseeki Core](../Daseeki-Core) — adds the bag's options panel to the shared Daseeki options hub. Daseeki Bags works standalone without it.
